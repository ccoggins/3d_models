# Steps

Make lithophane as square as possible, taking note of side length. Use the actual side length, e.g. if in makerworld lithophane maker, you select 100 mm for the picture size, the actual print will be ~104 mm on a side. Set this value for `pic_width` in all files.

Print four frame_post.scad posts

Print the base. 

base.scad is for use with lamps that mount with a threaded rod, e.g.

https://www.amazon.com/dp/B0DMCXZHJJ?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1

base2.scad is for use with lamps that use spring holders, e.g.

https://www.amazon.com/dp/B09QRV9LR5?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1

Print the top and bottom of the box with frame.scad. Use center_cutout=false; & diamond_cutout=true; for the top. Use center_cutout=true; & diamond_cutout=false; for the bottom. 

Screw the four posts to the bottom using #6 1/2" wood screws. Add the four lithophanes, textured side facing out. Screw on the top. Add the lamp to the base. The hole in the bottom should fit over the lamp.

https://www.homedepot.com/p/6-x-1-2-in-Zinc-Plated-Phillips-Flat-Head-Wood-Screw-100-Pack-822712/317479388
